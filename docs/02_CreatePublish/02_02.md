## Task 02: Test the agent

## Description
You'll run a test of the Order Processing Agent tool from the Instructions section and monitor the Activity pane and test machine interface as the agent reads the Excel workbook, submits orders to the Order Management Portal, and writes the generated Order IDs back to the spreadsheet. You must be available to approve any MFA prompts during the test.

## Success criteria
- You initiated a test from the Instructions section and monitored the agent processing orders in the test machine interface.
- You confirmed the agent opened the Order Management Portal and the SharePoint site, submitted at least one order, and updated the spreadsheet with the Order ID and status.
- You selected Finish Testing to close the test session.

{: .warning }
> The agent connects to an internal SharePoint site that requires authentication. You must monitor the activity window during testing so that you can complete multi-authentication (MFA) if prompted. Testing will be suspended when the system is waiting for MFA, and the process will time out if you do not complete MFA.
>
> You may also receive an email notification to approve the test. The email is sent to the email address that you configured in Task 01. You must approve the request for the test to continue.
