## Task 03: Publish the agent and add a channel

## Description

You'll publish the Order Processing Agent to make it available to others, add the Microsoft 365 and Microsoft Teams channel, and submit the agent to the org catalog so that an administrator can approve its distribution to all users.

## Success criteria
- You published the agent using the Publish command and confirmed the process completed.
- You added the Microsoft 365 and Microsoft Teams channel and selected Add channel.
- You submitted the agent to the org catalog using Submit to org catalog and confirmed the submission dialog.


{: .note } 
> Publishing agents from trial accounts is no longer permitted. The user must have a valid Copilot license.

---

#### 01: 

1. On the command bar, select **Publish**.
	
	![rvh458bh.png](../../media/rvh458bh.png)

1. In the confirmation dialog, select **Publish**. Wait for the publishing process to complete.

	![1chq7z0y.png](../../media/1chq7z0y.png)

1. On the command bar, select **Channels**.

	![s5x20gt7.png](../../media/s5x20gt7.png)

1. In the list of tiles that display, select **Microsoft 365 and Microsoft Teams**.

	![lcuic8dt.png](../../media/lcuic8dt.png)

1. In the **Microsoft 365 and Microsoft Teams** pane, select **Add channel**.

	![wbphov2g.png](../../media/wbphov2g.png)
	![n8tajheq.png](../../media/n8tajheq.png)

1. In the **Microsoft 365 and Microsoft Teams** pane, select **Availability options**.

    ![5639dnj9.png](../../media/5639dnj9.png)

1. In the **Show in the store** section, select **Show to everyone in my org**.

	{: .warning } 
	> This action sends an approval request to an administrator. Before the agent becomes visible to everyone, the administrator must approve the request. You will approve the request in the next exercise.

	![31kmim0e.png](../../media/31kmim0e.png)

1. In the **Show in Teams store for org** pane, select **Submit to org catalog**.

	![8dr2iycl.png](../../media/8dr2iycl.png)

1. In the confirmation dialog, select **Yes**.

	![mxvzpyc7.png](../../media/mxvzpyc7.png)
